1. 数组的检测方法

ES5的isArray函数
function isArray (obj) {
    return Array.isArray(obj);
};

缺点：IE8之前的版本并不支持该方法

对象自身的constructor属性
function isArray (obj) {
    return (typeof obj !== 'undefined' && obj && obj.constructor === Array);
};
##### instanceof操作符 instanceof操作符用于判断某个构造函数的**prototype**属性是否在另一个被检测对象的原型链上，即判断instanceof**前面的对象**是否是**后面的类或对象**的**实例**。 
arr instanceof Array

缺点：constructor和instanceof在多个frame中来回执行时，跨frame实例化的对象彼此间并不共享原型链，因此这两种方法会失效。

对象原生的toString方法

Object.prototype.toString的行为：取得对象一个内部属性[[Class]]，根据该属性返回形如“[object Array]”的字符串。因此可利用call访问到对象的内部属性[[Class]]，把类型检测转换为字符串的比较。


isArray = function (obj) {
    return Object.prototype.toString.call(obj) == “[object Array]”;
};

其他方法

通过判断是否是拥有join方法以及length长度属性的对象来检测数组


function isArray (arg) {
    if (typeof arg === “object” &&
       (“join” in arg && typeof arg.join === “funciton”) &&
       (“length” in arg && typeof arg.length === “number”)) {
        return true;
    }
};

最佳检测方法

不管Array.isArray方法是否能够使用，都可以回到原生对象的toString检测方法上。

2.数组的sort()方法

基本使用

sort()方法对数组的元素进行排序，并返回原数组。默认情况下，没有传递参数时，对数组项进行升序排列。另外方法可接受一个比较函数compareFunction作为参数。

  • compareFunction(a, b)返回值小于0：a < b
  • compareFunction(a, b)返回值等于0：a = b
  • compareFunction(a, b)返回值大于0：a > b

    // 升序排列
    function ascSort(a, b) {
      if (a < b) {
    return -1;

      } else if (a > b) {
    return 1;

      } else {
    return 0;

      }
    };

    return -1;

    return 1;

    更简单的比较函数
    对于数字类型或valueof（）方法返回数字类型的对象类型，可使用更简单的比较函数。

对象数组的单属性排列


// 根据desc参数值，对数组按指定的property属性进行升降序排列
function objectSort (property, desc) {
    if (desc) {
        return function (a ,b) {
            return (a[property] > b[property]) ? -1 : (a[property] < b[property]) ? 1 : 0;
        }
    return function (a, b) {
        return (a[property] < b[property]) ? -1 : (a[property] > b[property]) ? 1 : 0;
    }
};
